## Shape-Specific Properties

| Property | Description |
| -------- | ----------- |
| **Email**  |[Email](common/Email.md)|
| **Deferred Time** | Default value: 0 Minute(s)<br>Specifies the amount of additional time that should be allowed to pass before the email notification is sent.<br>NOTE: If the Length sub-property is 0, the email notification will be sent without any delay. |
